# Starweb Libraries

**Starweb Libraries** is a long-term personal exploration of web engine architecture. It prioritizes small, composable libraries with experimentation as a core principle.

> This is not intended to be a publicly released framework, production engine, or drop-in library. It's an experimental lab for personal web development.

## Repositories

### Core
- [`engine`](https://github.com/starweb-libs/engine) — Canvas, input, game loop, and asset utilities
- [`physics`](https://github.com/starweb-libs/physics) — SAT and circle collision detection
- [`tween`](https://github.com/starweb-libs/tween) — Frame-driven tween manager with easing
- [`audio`](https://github.com/starweb-libs/audio) — Web Audio API playback and management
- [`ui`](https://github.com/starweb-libs/ui) — Canvas UI primitives for browser games

## Development
- [`template-library`](https://github.com/starweb-libs/template-library) — Scaffold template for new library projects

### Games
- [`pixel-parker`](https://github.com/masonlet/pixel-parker) — Top-down vehicle parking puzzle game
- [`starweb-sandbox`](https://github.com/masonlet/starweb-sandbox) — Stress test and sandbox environment
